This folder contains two sub-folders.
The sub-folder python_scripts contains the python scripts used: 
	* for the lncRNA extraction and analysis pipeline, 
	* the sncRNA extraction,
	* others analysis (the lncRNAs present in both transcriptomes and the script to get the fasta sequences of the transcripts for a TrinityStats analysis)

The sub-folder R_scripts contains the scripts use to make the average of the replicates and their export into excel file and the heatmaps
